aggregated_by_line_has_age_gender.csv

each line in the file represents a patient with the following columns:
patientID,gender,age,(date,CIDcode)

example:
10000002,'F',7.8932238193,('2014-01-13', 'H91'),('2014-01-14', 'H90'),('2014-01-22', 'H52'),('2014-02-14', 'H02')

patientID: 10000002
gender: ‘F’
age:7.8932238193
record_history (date,CIDcode): ('2014-01-13', 'H91'),('2014-01-14', 'H90'),('2014-01-22', 'H52'),('2014-02-14', 'H02')
